# Career Path Builder

מטרת הריפו:
לבנות מסלול למידה וקריירה אחיד לקבוצת פיתוח.

## איך עובדים?
1. משתמשים בריפו כטמפלייט
2. מקיימים דיון קבוצתי וממלאים את הקבצים בתיקיות
3. מתכנסים להסכמות קבוצתיות
4. מייצרים קובץ FINAL אחד

## תוצר סופי:
1. קובץ אחד שמייצג את-PATH של הקבוצה.
   תיעזרו בקובץ milestones.md להסכמה קבוצתית על התוכן במסלול הלמידה.
3. קובץ אחד שמייצג את ה-CONTRACT של הקבוצה
   תיעזרו בקובץ contract.md כדי לייצר חוזה איכותי שנוגע בכל הנקודות שיכולות להוות בעיה במהלך פרויקט משותף.
